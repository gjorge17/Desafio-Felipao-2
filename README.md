Calculadora de Partidas Rankeadas

Descrição do Projeto

Este projeto foi desenvolvido como parte do desafio de código do Bootcamp da DIO Santander 2025 - Fundamentos de Lógica de Programação.
O objetivo principal é consolidar conhecimentos sobre lógica de programação, criando um sistema que classifica o nível de um herói com base em seu saldo de vitórias e derrotas.

Diferente de uma implementação simples, este projeto conta com uma interface de linha de comando (CLI) interativa, permitindo múltiplas simulações sem a necessidade de reiniciar o programa.

Funcionalidades

- Cálculo de Saldo: Subtrai derrotas de vitórias para obter o saldo real.
- Sistema de Ranking: Classifica o herói desde o nível "Ferro" até "Imortal" baseado em faixas de pontuação.
- Interatividade: O usuário insere os dados via terminal.
- Loop de Execução: O sistema pergunta se o usuário deseja realizar um novo cálculo, mantendo o programa ativo.
- Validação de Dados: Tratamento básico para garantir que as entradas sejam numéricas.
